#Sequencer

Perform a sequence of events at tempo

Also see: [midi-sequencer](http://github.com/arirusso/midi-sequencer)

## Installation

`gem install simple-sequencer`
  
  or with Bundler, add this to your Gemfile
  
`gem "simple-sequencer"`
